Error Field Penetration Threshold Calculator

Welcome

This tool calculates the empirical error field penetration threshold δ using Monte Carlo uncertainty propagation on published multi-machine scalings.

Scalings

  • 2026 O,L OLS / WLS — E. M. Bursch et al. 2026, submitted to PPCF
  • 2020 O,L,H WLS / O,L WLS — N. C. Logan et al. 2020, Plasma Phys. Control. Fusion 62 084001

Single-Point Mode

Enter the six plasma parameters manually, pick a scaling & distribution, then click Calculate. Each calculation is added to the plot so you can overplot multiple cases (e.g. ITER vs. SPARC). Click Clear Plots to reset.

Batch (CSV Upload) Mode

Upload a .csv file. The first column must be a label or shot number; the remaining six columns are the plasma parameters. Column headers are matched case-insensitively.

Required CSV format

Exponent Editor

The right-hand panel shows the scaling exponents for the selected scaling. Editing any value automatically switches to Custom mode. You can save custom scalings (stored in your browser) and load / delete them later.

Plot Style

Click Plot Settings ⚙ to customise fonts, colours, axis labels, line widths, and figure size. Changes apply to the next render.

Exporting

  • Save Plot (PNG) — downloads the current figure as a high-resolution PNG.
  • Save Plot Data (CSV) — downloads every plotted distribution (bin centres + PDF) plus all input parameters and results.
  • Export Results to CSV (batch mode) — downloads the results table.
